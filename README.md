# Laravel Learning Roadmap

A simplified checklist of the main things to learn when becoming a
Laravel developer.

## 1. Fundamental PHP

-   Syntax dasar (variable, array, loop, condition)
-   Function
-   OOP (class, object, inheritance, interface, trait)
-   Namespace
-   Composer & PSR-4 autoloading

## 2. Laravel Basics

-   Struktur folder Laravel
-   Routing
-   Controller
-   Blade templating
-   Middleware
-   Environment config (.env)
-   Artisan commands

## 3. Database

-   Migration
-   Schema design
-   Seeder
-   Factory
-   Query Builder
-   Database transactions

## 4. Eloquent ORM

-   Model
-   CRUD dengan Eloquent
-   Relationship
-   Eager loading
-   Accessor & Mutator
-   Mass assignment

## 5. Form & Validation

-   Form handling
-   FormRequest validation
-   Error handling
-   Redirect & session flash

## 6. Authentication & Authorization

-   Login / register
-   Session authentication
-   Sanctum
-   Middleware auth
-   Policies & Gates

## 7. Clean Architecture

-   Service layer
-   Repository pattern
-   SOLID principle
-   Thin controllers

## 8. API Development

-   REST API
-   JsonResource
-   API pagination
-   API authentication
-   Rate limiting

## 9. Async & Background Processing

-   Jobs
-   Queues
-   Redis
-   Task scheduling

## 10. Production & Deployment

-   Environment configuration
-   Caching
-   Queue workers
-   Logging
-   Deployment (Nginx / Docker / CI/CD)

------------------------------------------------------------------------

## Recommended Learning Order

PHP → Laravel Basics → Database → Eloquent → Validation → Auth → Clean
Architecture → API → Queue → Deployment
